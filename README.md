# Usage

## Compatibility

- ArchLinux [ApolloLinux]
- Fedora (soon) [Aurora, Zirconium, Bluefin, Silverblue]
- AlmaLinux (soon) [Alma-Bootc]

1. Write files in yaml format to /etc/bottomtext.d/*.yaml *(check conf-exemple)*
2. Run `sudo bottomtext`
3. Your packages are installed/updated or removed


# Building

`mise install`

`just compile` or `mise tasks r compile`

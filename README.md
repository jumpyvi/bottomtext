# Usage

1. Create the store: `sudo mkdir -p /etc/bottomtext/ && sudo touch /etc/bottomtext/store.txt`
2. Add a package: `sudo bottomtext add ghcr.io/jumpyvi/helix:latest`
3. Sync & merge packages in your store: `sudo bottomtext sync`


# Building

`mise install`

`just compile` or `mise tasks r compile`
